# Tarkony-express-backend

Tarkony rövid leírása
Tervezet szintjén leírás Tarkony ra vonatkozó Backend funkcionalitása stackek bevonásával

## Used stack

"dependencies": {
"express": "^5.1.0"
},
"devDependencies": {
"@tsconfig/node22": "^22.0.2",
"@types/express": "^5.0.5",
"@types/node": "^24.9.2",
"tsx": "^4.20.6",
"typescript": "^5.9.3"
}

## Install and Run

npm install

Build watch:
npx tsc --watch

Run / reset:
npx tsx --watch src/index.ts
